# Input & post-processing tools for VASP and LAMMPS 

This project consists of 3 files:
* Pre-POSCAR is generated for adding point defect struture randomly on a POSCAR
* mlip-prep is a tool to create configure file suitable to use on MLIP-3 (usally done by built-in command but didn't suitable for VASP 6.5)
* post-proces file can be used a initial point to learn post-processing. Basic functions for VASP and LAMMPS output: RDF (radial distribution function), E-V vs step, Equation of State and etc
